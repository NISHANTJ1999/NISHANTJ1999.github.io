SOS
Before every Himalayan or remote ride: File a route plan with a trusted contact. Carry a PLB (Personal Locator Beacon) or satellite communicator. Know the nearest BSNL tower location — other networks drop fast above 3,000m. Keep the State Disaster Management Authority number of each state you cross.
